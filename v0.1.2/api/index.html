Checkerboard.CheckerboardMatrix — Type
CheckerboardMatrix{T<:Union{AbstractFloat, Complex{<:AbstractFloat}}}

A type to represent a checkerboard decomposition matrix.

Fields

  • transposed::Bool

    Is checkerboard matrix transposed.

  • inverted::Bool

    Is checkerboard matrix inverted.

  • Nsites::Int64

    Number of sites in lattice/dimension of checkerboard matrix.

  • Nneighbors::Int64

    Number of neighbors.

  • Ncolors::Int64

    Number of checkerboard colors/groups.

  • neighbor_table::Matrix{Int64}

    Neighbor table represented by a (2,Nneighbors) dimensional matrix, where each column contains a pair of neighboring sites.

  • coshΔτt::Vector{T} where T<:Union{AbstractFloat, Complex{<:AbstractFloat}}

    The $\cosh(\Delta\tau t)$ values.

  • sinhΔτt::Vector{T} where T<:Union{AbstractFloat, Complex{<:AbstractFloat}}

    The $\sinh(\Delta\tau t)$ values.

  • perm::Vector{Int64}

    The checkerboard permutation order relative to the ordering of the original neighbor table.

  • colors::Matrix{Int64}

    The bounds of each checkerboard color/group in neighbor_table.

Checkerboard.checkerboard_decomposition! — Function
checkerboard_decomposition!(neighbor_table::Matrix{Int})

Given a neighbor_table, construct the checkerboard decomposition, which results in the columns of neighbor_table being re-ordered in-place. Two additional arrays are also returned:

  • perm::Vector{Int}: The permutation used to re-order the columns of neighbor_table according to the checkerboard decomposition.
  • colors::Matrix{Int}: Marks the column indice boundaries for each checkerboard color in neighbor_table.
